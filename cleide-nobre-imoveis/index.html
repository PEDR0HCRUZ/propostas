01 · contexto

O projeto

Cleide precisa de um site para catalogar e divulgar oportunidades de imóveis para locação e venda.

A ideia é criar um site limpo, com foco maior nos imóveis disponíveis. A ação principal do site será enviar o usuário para o whats app, levando o link e/ou código de referenciação do imóvel desejado que ele escolheu no site.

Será feito também um painel para cadastrar e remover imóveis de forma simples e clara para fácil utilização no dia a dia.
